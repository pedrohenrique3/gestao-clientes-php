# gestao-clientes-php

Sistema de gestão de clientes de empresa fictícia de provedor de internet

Este projeto tem por objetivo treinar o que eu aprendi nas aulas iniciais de PHP. Ainda sei apenas o básico, portanto, o projeto é condizente com minha capacidade. Trata-se de um sistema de gestão de clientes para uma empresa fictífica de internet. Ainda estou preparando o php, e a css do sistema será feita quando eu completar as funcionalidades do sistema.
Atualmente só as páginas de login do cliente e do usuário foram criadas. Funcionalidades futuras serão desenvolvidas nos próximos dias.

Este projeto é o primeiro antes de eu começar a estudar programação orientada a objeto. Parte do sistema de login referente a verificação dos dados foi construida com base em alguns tutoriais, pois vi que somente com o que eu sabia, o sistema apresentava falhas.
O banco de dados foi criado totalmente por mim, e os dados de cpf, nome, endereço... etc., são totalmente fictífios e criados por meio de geradires automáticos na internet.

Próximos passos

-busca e visualização dos clientes pelo admnistrador


Aceito dicas, sugestões, pois meu objetido é treinar e aprender.
